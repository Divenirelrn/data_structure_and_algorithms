
动态规划题目类型：
    1.计数
        有多少种方法走到右下角
        有多少种方法选出k个数使得和为sum

    2.求最大最小值
        从左上角走到右下角路径的最大数字和
        最长上升子序列长度

    3.求存在性
        取石子游戏，先手是否必胜
        能不能选出k个数使得和为sum


动态规划组成部分：
    1.确定状态
        最后一步
        子问题

    2.转移方程
        f[X] = min{f[X-2] + 1, f[X-5] + 1, f[X-7] + 1}

    3.初始条件与边界情况
        X-2,X-5,X-7小于0：定义f[-1]=f[-2]=...=正无穷
        什么时候停下来：f[1] = min{f[-1] + 1, f[-4] + 1, f[-6] + 1} = +Inf, 表示拼不出来

        初始条件： f[0] = 0

    4.计算顺序
        大多数是顺序的
        计算f[x]时，等式右边全都计算过了


时间复杂度：
    n * m
